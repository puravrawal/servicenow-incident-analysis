# IT Service Desk Incident Analysis (ServiceNow-style Data)

This project analyzes IT Service Desk performance using a ServiceNow-style incident event log dataset, with a focus on data transformation, SLA metrics, and operational insights.

- Description: Event-based incident lifecycle data with multiple records per incident

## Project Overview
- Transformed raw event-level logs into an incident-level analytics dataset
- Built a SQL ETL pipeline in MySQL to prepare data for analysis
- Cleaned inconsistent timestamps and corrected negative resolution times
- Designed business KPIs such as average resolution time and SLA breach rate
- Built Tableau dashboards to visualize IT Service Desk performance

## Tools Used
- MySQL (SQL, window functions)
- Tableau
